📖 คู่มือขอรับรหัสผ่านสำหรับแอป (App Password Setup Guide)

ขั้นตอนการรับรหัสผ่านแอป 16 หลักสำหรับ Gmail (Google):

  1. เข้าสู่หน้าจัดการบัญชี Google Account ของคุณ
  2. เลือกเมนู "ความปลอดภัย" (Security) ในแท็บด้านซ้าย
  3. ตรวจสอบให้แน่ใจว่าได้เปิดใช้งาน "การยืนยันสองขั้นตอน" (2-Step Verification) แล้ว
  4. คลิกเข้าไปที่เมนู "การยืนยันสองขั้นตอน" เลื่อนลงมาล่างสุด แล้วกดเลือกปุ่ม "รหัสผ่านสำหรับแอป" (App passwords)
  5. ป้อนชื่อจำแนกแอปพลิเคชัน (เช่น TicketSolve SMTP) แล้วคลิกปุ่ม "สร้าง" (Create)
  6. ระบบ Google จะแสดงรหัสผ่านแอปสุ่มขึ้นมาความยาว 16 ตัวอักษร
  7. คัดลอกรหัสผ่านนี้ไปป้อนลงในช่อง รหัสผ่านแอป ของหน้าเว็บ TicketSolve

ขั้นตอนการรับรหัสผ่านแอป 16 หลักสำหรับ Outlook / Office 365 / Microsoft 365:

  1. เข้าสู่หน้าจัดการความปลอดภัยบัญชี Microsoft Account Security
  2. ลงชื่อเข้าใช้งาน และเลือกเมนู "ตัวเลือกความปลอดภัยขั้นสูง" (Advanced security options)
  3. ตรวจสอบให้แน่ใจว่าได้เปิดใช้งาน "การตรวจสอบสองขั้นตอน" (Two-step verification) แล้ว
  4. เลื่อนลงไปด้านล่างจนเจอหัวข้อ "รหัสผ่านของแอป" (App passwords) จากนั้นคลิกปุ่ม "สร้างรหัสผ่านแอปใหม่" (Create a new app password)
  5. ระบบ Microsoft จะทำการสุ่มรหัสผ่านความปลอดภัยความยาว 16 หลัก ให้ทันที
  6. คัดลอกรหัสผ่านนี้ไปป้อนลงในช่อง รหัสผ่านแอป ของหน้าเว็บ TicketSolve
